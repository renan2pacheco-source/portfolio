# Currículo — Renan Nunes Pacheco

> Site/currículo profissional sério, com foco em habilidades, ferramentas e resultado para empresas. Hospedado no GitHub Pages.

🌐 **URL produção:** `https://renan2pacheco-source.github.io/portfolio/`

## Código

- **Projeto:** `/home/renan/projetos/portfolio/`
- **Conteúdo principal:** `src/data/content.ts`

## Stack

| Camada | Tech |
|---|---|
| Framework | Astro 5 |
| UI interativa | React 19 |
| Animações | Framer Motion |
| Estilo | Tailwind CSS 4 |
| Linguagem | TypeScript |
| Ícones | Simple Icons CDN + Lucide |
| Fonte | Geist + Geist Mono |
| Deploy | GitHub Actions → Pages |

## Estrutura do currículo

1. Hero profissional com localização correta: **Vale do Anari, RO — Brasil**
2. Resumo profissional sério
3. Experiências: suporte técnico, vendas, freelancer autônomo, personal trainer e rotina operacional
4. Ferramentas e habilidades com ícones oficiais
5. Como Renan pode gerar resultado para a empresa
6. Contato profissional via WhatsApp/email

## Como rodar

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Diretrizes de conteúdo

- Não tratar o site como vitrine de projetos pessoais.
- Não vender Renan como desenvolvedor profissional.
- Foco em resultado: suporte, atendimento, produtividade, organização, vendas, automação e melhoria de fluxo de trabalho.
- Manter tom sério e profissional.
